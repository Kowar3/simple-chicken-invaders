# 🚀 Space Invaders (Pygame)

Jednostavna 2D igra inspirisana Space Invaders, napravljena u Python-u koristeći Pygame biblioteku.

## 🎮 Opis

Igrač kontroliše svemirski brod i pomera se levo-desno kako bi pucao na neprijatelje.  
Cilj igre je pogoditi što više neprijatelja i osvojiti što veći broj poena.  
Igra se završava kada neprijatelji dodirnu igrača.

## 🛠️ Tehnologije

- Python
- Pygame

## ▶️ Pokretanje

1. Instaliraj Pygame:
```bash
pip install pygame
```

2. Pokreni igricu:
```bash
python main.py
```

---

## 📁 Struktura

- `main.py` – glavna logika igre  
- `background.png` – pozadina igre  
- `space-invaders.png` – slika igrača  
- `play.png` – slika neprijatelja  
- `bullet.png` – slika metka  
- `ufo.png` – ikonica igre  
- `background.wav` – muzika u pozadini  
- `laser.wav` – zvuk pucanja  
- `explosion.wav` – zvuk pogotka  

---

## ⚙️ Funkcionalnosti

- Pomeranje igrača levo i desno  
- Pucanje metaka  
- Više neprijatelja na ekranu  
- Detekcija sudara
